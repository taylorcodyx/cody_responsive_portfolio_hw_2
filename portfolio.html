Random Password Generator

Given the need for extra security these days, this is a random password generator that will populate a strong password for the user that is anywhere from 8-128 characters long. Upon clicking the "Generate Password" button, the user will be prompted with desired criteria for the password such as including numbers, upper case letters, lower case letters, and special characters in the generated password. Note: you must select ONE of the above options. After choosing how long the user wants the password to be (8-128 characters,) and once all the parameters have been defined, a password will then appear in the password box. Utilized with HTML5, CSS, Javascript
